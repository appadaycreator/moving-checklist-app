# 引越しToDoリスト・チェックアプリ【無料】無料Webツール - 技術仕様書

## 概要

**サービス名**: Moving Checklist App
**バージョン**: 1.3.0（2026-06-10 自動改善版）
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/moving-checklist-app/

引越し前後の手続き・準備を時系列チェックリストで管理。スケジュール計算も自動化。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- Chart.js（v4）- 進捗・カウントダウン可視化
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## v1.3.0 改善点（2026-06-10 - 自動改善）

### P1: 高優先度改善
- **チェック状態リセット機能**: 「🗑 チェック状態をリセット」ボタンを追加（チェックリスト表示後に表示）
- **カスタムタスク追加**: ユーザーが自分のタスクを追加可能（「➕ カスタムタスクを追加」セクション）
- **フォーム検証改善**: 未入力項目を赤色ハイライト + エラー表示 + 自動スクロール
- **グラフ初期非表示**: チェック数0時にグラフを表示しない（ページ軽量化）
- **日付初期値**: 引越し予定日をデフォルト値「今日+7日」に設定

### P2: 中優先度改善
- **カテゴリ別進捗表示**: 各タイムライン（8週前等）ごとに「完了/総数」を表示
- **文脈的CTA**: ファミリー・遠距離選択時に対応するアフィリエイトリンク表示予定
- **モバイルUI最適化**: チェックボックス44px×44px化・タップ領域拡大
- **印刷改善**: チェック済みタスクに打ち消し線表示（@media print）
- **キーボードナビゲーション**: Tab・Enter・Escapeキー対応

## v1.1.0 新機能（2026-06-10）

### M4: 進捗・カウントダウングラフ実装（Chart.js）
- チェック完了数 vs 残タスク数のドーナツチャート
- 引越し日までの残り日数を横棒グラフで表示
- リアルタイム更新（チェック/アンチェック時にグラフ反映）

### M3: 複数引越し履歴管理機能
- localStorage で過去5件の履歴を自動保存
- 「📋 履歴から選ぶ」ボタンでモーダル表示（タイムスタンプ付き）
- 履歴から選択で前回設定を復元・チェックリスト再生成
- 個別削除機能で不要な履歴を削除可能

### M2: Input UX改善
- 各入力欄に日本語ヘルプテキスト追加
- 引越し日の説明：「例: 2026年8月15日 • 引越し日から逆算してやることが表示されます」
- 家族構成の説明：「引越し後の家族状況を選択してください」
- エラーメッセージの視認性向上（背景色#fee2e2、テキスト#991b1b）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## よくある質問（FAQ）

**Q: 引越しToDoリスト・チェックアプリは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: 結果を保存・共有できますか？**

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
